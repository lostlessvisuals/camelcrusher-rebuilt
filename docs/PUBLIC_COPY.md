## README / Repo Intro

Short version:

`CamelCrusher Rebuilt` is a native Apple Silicon remake of CamelCrusher for modern macOS. It includes `AU`, `VST2`, and `VST3` builds, with `VST2` specifically aimed at reopening older projects that used CamelCrusher.

Longer version:

CamelCrusher was a staple effect for a lot of Mac users, but on modern macOS, and especially on Apple Silicon, the original plug-in became harder and harder to keep alive. `CamelCrusher Rebuilt` brings that workflow back with native Apple Silicon builds for modern Mac hosts, while preserving the old `VST2` identity closely enough to reopen previous projects that depended on it.

## Release / Post Copy

`CamelCrusher Rebuilt` brings CamelCrusher back to Apple Silicon Macs with native `AU`, `VST2`, and `VST3` builds. The main goal is simple: old sessions should open again, and modern macOS users should have a native way to keep using the effect without depending on the original abandoned setup.
